# 1페이지 포트폴리오 기능 명세서 (PRD)

## 1. 프로젝트 개요
* **목적:** 모던하고 전문적인 개인 포트폴리오 1페이지(One-page) 웹사이트 구축
* **타겟 화면:** PC, 태블릿, 모바일 (모바일 우선 반응형 웹)

## 2. 공통 기능 요구사항
* **네비게이션 (GNB):** 상단에 고정(Sticky)되며, 스크롤을 내리면 배경이 약간 투명해집니다. 메뉴 글자를 클릭하면 해당 구역으로 부드럽게 스크롤되어 이동합니다(Smooth Scroll).
* **반응형 메뉴:** 스마트폰 화면에서는 상단 메뉴가 '햄버거 버튼(☰)'으로 바뀌고, 누르면 옆에서 서랍처럼 메뉴판이 밀려 나옵니다(shadcn/ui의 Sheet 부품 활용).
* **테마 토글:** 사이트를 밝게 볼지, 어둡게 볼지 선택하는 '다크 모드/라이트 모드' 버튼을 제공합니다.

## 3. 섹션별 세부 기능 명세
| 구분 (섹션명) | 화면 구성 요소 | 세부 기능 및 상호작용 (인터랙션) |
| :--- | :--- | :--- |
| **1. Hero (도입부)** | 메인 카피, 서브 카피, 대표 이미지(또는 3D 오브젝트), 액션 유도 버튼(CTA) | - 사이트 접속 시 글자와 이미지가 부드럽게 떠오르는 애니메이션(Fade-in)<br>- '작품 보기' 버튼 클릭 시 Portfolio 섹션으로 자동 스크롤 이동 |
| **2. About (소개)** | 자기소개 글, 보유 기술(Skills) 태그 목록 | - 기술 태그에 마우스 포인터를 올리면 색상이 반전되거나 살짝 커지는 강조 효과 |
| **3. Portfolio (작품)** | 작품 썸네일(미리보기) 사진 카드들, 카테고리 필터 | - 전체/디자인/코딩 등 카테고리 버튼 클릭 시 해당하는 작품 카드만 부드럽게 걸러져서 화면에 재배치됨<br>- 특정 작품 카드를 클릭하면 화면 중앙에 팝업창(Dialog)이 열리며 상세 사진과 설명을 보여줌 |
| **4. Experience (이력)** | 연도별 경력 및 학력 타임라인 | - 사용자가 마우스 스크롤을 내려서 이 구역이 화면에 보일 때, 시간 순서대로 내용이 차례차례 나타나는 시각적 효과 |
| **5. Contact (연락처)** | 이메일 주소, SNS 링크, 문의 남기기 폼(Form) | - 폼에 이름이나 이메일 형식을 잘못 적고 '전송'을 누르면 폼 아래에 붉은색 에러 메시지 즉시 노출<br>- 정상 전송 시 화면 구석에 '전송 완료' 알림창(Toast)이 예쁘게 떴다가 3초 뒤 스르륵 사라짐 |

## 4. 비기능 및 디자인 요구사항
* **디자인 톤앤매너:** 고급스럽고 정갈한 무채색(블랙, 다크 그레이) 위주의 배색을 기본으로 사용합니다.
* **사용 기술:** Next.js, Tailwind CSS, shadcn/ui 세 가지를 완벽하게 조합하여 구성합니다.
